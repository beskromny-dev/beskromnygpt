# Источники для утреннего дайджеста «Бескромный»

Этот список — для Manus (или любого другого AI-агента), чтобы он тоже мог собирать утренний новостной обход параллельно с нашей системой.

## Инструкция для Manus

Каждое утро в 6:00 МСК пройдись по источникам ниже и собери новостной бриф за последние 24 часа. Формат брифа:

1. **Выжимка (4 абзаца):** главное из свежего с цифрами + история дня + жемчужина + финальная строка «Плюс N идей для постов»
2. **Полный бриф:** Сводка / История дня / Дни рождения / Новости дня (10-12 пунктов) / Жемчужина / Рекомендация на день

Приоритет отбора: **абсурд и парадокс > конкретные цифры > неочевидные связи**.

Баланс категорий в блоке «Новости дня»: 3 бизнес + 2-3 технологии/ИИ + 3 психология/общество + 2-3 культура/лайфстайл.

Жёсткие фильтры (НЕ включать): чистую политику без бизнес-призмы, локальный PR без цифр, криминальную хронику, узкотехнические релизы SDK/фреймворков.

Ключевые концепции автора: «ИИ — экзоскелет для мозга», «Калорийный контент», «Линди-навыки», «Двухфакторная авторизация», «Территория любви».

Формат результата: PDF с кликабельными ссылками + короткая выжимка текстом.
